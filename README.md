# Ejercicios de Pilas en C#

## 1. Verificación de paréntesis balanceados
Programa que analiza una expresión matemática y determina si los paréntesis, llaves y corchetes están correctamente balanceados.

### Ejemplo:
Entrada: `{7 + (8 * 5) - [(9 - 7) + (4 + 1)]}`  
Salida: `Fórmula balanceada.`
